# ⭐ Text Sentiment Analyzer – NLP & Machine Learning Project

An end-to-end web application that predicts whether a given text is **Positive, Negative, or Neutral** using Natural Language Processing (NLP) and Machine Learning models.

---

## 📌 Overview
This project processes user input text using NLP techniques and converts it into numerical features using TF-IDF. It then applies machine learning models to classify the sentiment and returns the result along with a confidence score through a web interface.

---

## 🔧 Tech Stack
- Python, Flask  
- Scikit-learn (Logistic Regression, Naive Bayes, TF-IDF)  
- NLTK (text preprocessing, stopwords, stemming)  
- HTML, CSS  
- Pickle  

---

## 🚀 Features
- Real-time sentiment prediction  
- Text preprocessing and normalization  
- TF-IDF feature extraction  
- Confidence score for predictions  
- Simple and user-friendly interface  

---

## ⚙️ Workflow
1. User inputs text  
2. Text preprocessing (cleaning, stopword removal, stemming)  
3. Feature extraction using TF-IDF  
4. Model predicts sentiment  
5. Output displayed with confidence score
